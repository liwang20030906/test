## 🌟 关于项目

作为研究方向偏向环境健康的一名遥感专业的学生，分析数据基本是非常重复性质的工作，并且由于统计学知识的不完善，所以每次建模完成之后都需要把模型参数输入给GPT，效率低下，且生成报告相对复杂。所以，调用Coze的Agent智能体，设计了一个从上传数据-数据建模-分析数据-输出报告的自动化流程，极大提升效率。

## ✨ 特性

- **特性 1**: 科研工作者视角：直接拿到专业的分析报告，并且根据一些环境变量的波动进一步预测疾病发生率的变化，辅助政策制定。
- **特性 2**: 大众视角：看不懂分析报告专业数据，大模型直接转化成通俗易懂的大白话并且给出相应的出行建议。
- **高性能**: SSE流式响应低成本实现流式输出，降低等待时间；本地建模+AI分析，提升效率的同时降低AI幻觉的可能性。
- **易于扩展**: 可发展成为任意领域分析+报告的形式。
- **跨平台**: 支持 Windows, macOS, Linux 等。
